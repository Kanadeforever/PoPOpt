# 兼容性与支持矩阵

## 可执行文件

| 构建版本 | 检测方式 | 语音策略 |
|---|---|---|
| GOG / 类似未打包版本 | 无 `.bind` 或 `.extra` 标记 | 通过字节验证确认 GOG RVA。 |
| Steam 未打包版本 | `.extra` 节区 | 通过字节验证确认 Steam 未打包 RVA。 |
| 原始 SteamStub 打包版本 | `.bind` 节区 | 等待运行时解包，然后验证 Steam 运行时 RVA。 |
| 未知的已修改可执行文件 | 保守策略 | 跳过不匹配的危险补丁并记录结果。 |

## 当前测试中已验证的行为

- 文本语言注册表拦截在 `DllMain` 返回前安装时可正常工作。
- 通过四点事务性补丁实现英语语音搭配中文文本。
- SteamStub 运行时语音补丁已在早期稳定分支中验证，并在模块化模块中保留。
- 集成的 TPF 加载器使用 `top-compact-complement` 模式成功匹配并替换了提供的 `XBOX.tpf` 中所有十个控制器提示哈希值。
- DirectInput Win 键恢复和焦点丢失鼠标释放在窗口/无边框模式下可正常工作。

## 未包含的功能

- 菜单过渡加速。
- XInput 热插拔恢复。
- 永久的游戏 EXE 修改。
- TexMod.exe、`tmldr.dll` 或 `tmrls.dll`。
