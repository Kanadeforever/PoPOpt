# 贡献指南

1. 保持最终运行时产物为单一的32位ASI文件。
2. 将共享基础设施放在`src/core`中，功能行为放在`src/modules`中。
3. 通过`HookManager`注册IAT钩子；模块不得独立覆盖同一IAT槽位。
4. 确保模块初始化是幂等的。
5. 不要添加对DInput、D3DX、XInput、zlib DLL或编译器运行时DLL的硬依赖。
6. 保留事务性语音补丁的默认设置。
7. 为新设置添加配置验证和兼容性报告输出。
8. 提交更改钩子、RVA或资源匹配的拉取请求时，需包含日志和测试矩阵。
9. 不要提交游戏可执行文件、专有SDK可再分发文件或私人用户日志。
